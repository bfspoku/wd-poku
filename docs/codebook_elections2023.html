codebook_elections2023

Intro

Description of the dataset on the official provisional election results for the 2023 federal elections on opendata.swiss. Currently, only a provisional test data set with dummy results of the 2019 elections is available on opendata.swiss:

Test data set: https://ogdch-abnahme.clients.liip.ch/de/dataset/eidgenossische-wahlen-2023-testing

In the following, the characteristics of these files are described. For this purpose, 3 different states are provided as test data for each file:
1. empty files. State how the file looks like before the election day.
2. state, how the file looks like with partial results during the election day.
3. state how the file looks like at the end of the election.

Warning: The file structures of the JSONs here are currently NOT yet final. Any changes to these files will be published on this page and in the changelog on opendata.swiss.
